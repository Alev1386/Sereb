Исходные данные представляют собой строку символов, разделенных на предложения. Написать программу, включающую две процедуры, выполняющие следующие действия:
1. определение и вывод на печать предложения, в котором чаще других
встречаются цифры;
2. замену цифр, встречающихся в строке, на соответствующие этой цифре числительные.
Чтение данных их файла производить с использованием функций ввода/вывода языка С++
Алгоритм должен быть параметризован; обмен данными с подпрограммой должен осуществляться только через параметры; каждый из наборов исходных данных хранится в отдельном файле.
